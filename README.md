<h2>프로젝트 소개</h2>

<li>HTML, CSS, JavaScript로 만든 간단한 To DO List 입니다.  </li>

<h2>기능</h2>
  <li>할 일의 기록 및 완료, 삭제 기능</li>
  <li>전체, 해야 할 일 완료 한 일로 탭을 나누어 관리가 가능 </li>
